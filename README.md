# MakeItO

## Overview

MakeItO is a fabrication management system designed for metal and wooden fabrication businesses. It helps manage customer orders, project details, material records, and production workflows efficiently.

## Features

* Order Management
* Project Tracking
* Material Management
* Customer Record Management
* Simple and User-Friendly Interface

## Technologies Used

* Java
* Eclipse IDE
* MySQL (if used)
